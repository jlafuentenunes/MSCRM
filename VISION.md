# MsCRM - Visão e Roadmap do Produto 🚀🛸

## 🎯 Objetivo Principal
Criar um CRM premium e intuitivo para uso interno inicial, com o objetivo de escalar para um produto comercial de mercado. O foco está na **Experiência do Utilizador (UX)**, **Estética Premium** e **Automações Inteligentes**.

## 🏗️ Fases de Desenvolvimento

### Fase 1: Organização Interna (Atual)
- [x] Gestão de Leads e Clientes.
- [x] Sistema de Automações (Rule Engine).
- [x] Caixa de Correio Integrada (Webmail).
- [x] Sistema de Tickets e Suporte.
- [ ] Gestão de Projetos e Tarefas.
- [ ] Dashboard Financeiro Básico.

### Fase 2: Integração e Conectividade
- [ ] **Integração Cegid Primavera (v10 Web API):**
    - Sincronização de Entidades (Clientes).
    - Consulta de Documentos de Venda e Pendentes.
    - Transformação de Leads em Clientes no ERP.
- [ ] Webhooks para integrações externas (Zapier/Make).

### Fase 3: Escalabilidade e Comercialização
- [ ] Multi-tenancy (SaaS ready).
- [ ] Sistema de Subscrições e Pagamentos.
- [ ] App Mobile Nativa (React Native/Expo).

---
## 💡 Notas Técnicas
- **Frontend:** React + Tailwind (foco em animações e HSL colors).
- **Backend:** PHP REST API com autenticação JWT/Sessão.
- **Integrações:** Prioridade de integração via REST API com sistemas ERP (Primavera).
